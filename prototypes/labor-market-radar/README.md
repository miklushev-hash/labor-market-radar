# Локальный прототип Labor Market Radar

## Что это

Это демонстрационный прототип целевого Labor Market Radar.

Он показывает будущую связку:

- внешние рыночные сигналы;
- внутренний контур практик и компетенций КОРУСа;
- кадровые риски;
- рекомендации с указанием уровня основания.

Внутренние данные в прототипе демонстрационные. Они нужны для проверки сценария интерфейса, а не для кадровых решений.

## Как запустить

Открой файл:

- `index.html`

Можно также запустить файл:

- `Открыть_прототип_Labor_Market_Radar.cmd`

Прототип не требует установки зависимостей и работает локально в браузере.

## Как должны обновляться данные

Целевая схема обновления данных:

```text
регулярные выпуски
  -> структурированный слой данных
  -> radar_snapshot.json
  -> дашборд / прототип
```

Сводный радар текущей ситуации в этой схеме опционален. Он может использовать тот же структурированный слой, но не является обязательным промежуточным этапом между выпусками и дашбордом.

Техническая архитектура описана в:

- `../../docs/17_dashboard_technical_architecture.md`
- `../../docs/18_snapshot_extraction_rules.md`
- `../../docs/19_index_methodology.md`

## Что проверить в прототипе

1. Переключение режима между общим радаром и внутренним пилотом.
2. Фильтр практики.
3. Переход между вкладками `Радар`, `Сигналы`, `Внутренний контур` и `Вывод`.
4. Связку внешнего сигнала с риском по роли или компетенции.
