# Словари разметки

Эта папка хранит словари для разметки недельных данных.

## Зачем
- чтобы одинаковые сигналы всегда помечались одинаково;
- чтобы можно было сравнивать недели без ручной пересборки;
- чтобы позже перейти к полуавтоматике без смены логики.

## Что здесь лежит
- `topics.csv` — темы;
- `signal_types.csv` — типы сигналов;
- `industries.csv` — отрасли;
- `regions.csv` — регионы;
- `sentiment_scale.csv` — шкала тональности.

## Правило
Если появляется новый тип сигнала или новая тема, сначала добавляем его в словарь, потом используем в недельных данных.
