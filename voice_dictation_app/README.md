# Голосовая диктовка для Windows 10

Это простое приложение для записи речи в текст. Здесь есть два варианта:

1. Python-приложение с графическим интерфейсом.
2. Локальная HTML-страница для браузерного голосового ввода.

## Вариант 1: Python-приложение

Требования:
- Python 3.8+.
- Интернет для распознавания речи через Google.
- Микрофон.

Установка:
```bash
cd voice_dictation_app
python -m pip install -r requirements.txt
```

Запуск:
```bash
python voice_dictation.py
```

Как пользоваться:
- Нажми кнопку `Диктовать`.
- Говори по-русски после сигнала.
- Текст появится в поле.
- Скопируй, сохрани или очисти.

Если при установке `PyAudio` возникают проблемы на Windows, можно искать готовое колесо (`.whl`) или использовать другой метод.

## Вариант 2: Браузерная страница

Открой `voice_dictation.html` в Google Chrome или новом Microsoft Edge.

Лучше запустить локальный сервер из этой папки:
```bash
python -m http.server 8000
```

Затем открой в браузере:
```
http://localhost:8000/voice_dictation.html
```

Если браузер не поддерживает Web Speech API, попробуй Chrome или Edge.

## Что делать дальше

- Если нужен быстрый тест, попробуй сначала HTML-страницу.
- Если хочется работать локально, используй `voice_dictation.py`.
- Этот инструмент предназначен для ввода текста голосом и копирования его в VS Code.
