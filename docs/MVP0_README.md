# Архивная заметка о раннем пилоте

## Что это

Этот документ сохраняется как архивная справка о самом раннем пилотном состоянии проекта.

Он нужен не как рабочая инструкция, а как краткое объяснение, с чего начинался продукт:

- сначала проект был недельным пилотом;
- логика строилась вокруг одного недельного среза;
- часть правил еще не была формализована;
- месячный, квартальный, полугодовой, годовой и стратегический контуры появились позже.

## Что важно понимать сейчас

Текущий проект вырос из MVP0 и больше не работает по ранней схеме.

Сейчас актуальная модель уже включает:

- недельный, месячный, квартальный, полугодовой и годовой режимы;
- обязательный полный обход реестра источников;
- отдельный контур компаний стратегического наблюдения;
- отдельный контур стратегических событийных сигналов;
