# Этап 3. Хранение истории

## Цель

Сохранять недельные, месячные, квартальные, полугодовые и годовые периоды так, чтобы можно было сравнивать динамику во времени.

## Принцип хранения

- каждый период хранится отдельно;
- недельный слой хранит сырье, нормализацию, метрики и итоговый выпуск;
- месячный, квартальный, полугодовой и годовой слои хранят итоговые выпуски периода;
- операционная отметка по всем периодам параллельно ведется в `Хранилище.xlsx`;
- журнал стратегических событий ведется в листе `Журнал_событий` книги `Хранилище.xlsx`.

## Справочник значений

- значения тем, типов сигналов, регионов и отраслей берутся из `taxonomy`;
- `signal_strength` хранится по шкале от `1` до `5`.
